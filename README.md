
# AstroLoggy

A website for two long time friends build with [Astro](https://astro.build/)



## Run Locally

Clone the project

```bash
git clone https://github.com/Chrissdroid/AstroLoggy.git
```

Go to the project directory

```bash
cd AstroLoggy
```

Install dependencies (with pnpm)

```bash
pnpm install
```

Start the server

```bash
pnpm start
```
